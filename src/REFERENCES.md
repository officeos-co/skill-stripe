## API Coverage
- **Customers**: list, get, create, update, delete, search customers
- **Payments**: list, get, create payment intents; confirm and cancel payments
- **Charges**: list, get, create charges
- **Subscriptions**: list, get, create, update, cancel, pause subscriptions
- **Products**: list, get, create, update, archive products
- **Prices**: list, get, create prices (one-time and recurring)
- **Invoices**: list, get, create, send, void, finalize invoices
- **Refunds**: list refunds, create full or partial refunds
- **Payouts**: list, get, create payouts (standard and instant)
- **Balance**: get balance, list balance transactions
- **Webhooks**: list webhook endpoints, create webhook endpoint
